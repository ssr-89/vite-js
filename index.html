HERO

  1. npm create vite@latest

    Дать название проекту.

  2. npm i

  3. Удалить:

    • counter.js
    • typescript.svg или javascript.svg
    • vite.svg

    И в index.html, .js удалить все отсылки к файлам

    В .css удалить стили

  4. В src добавить папки scss и js

  5. npm add -D sass
  6. npm install normalize.css

    В .scss прописать @import "normalize.css";
